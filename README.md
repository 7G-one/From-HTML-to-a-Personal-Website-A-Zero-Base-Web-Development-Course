# 🌐 HTML/CSS/JavaScript 板书教学课程

<h3 align="center"><a href="README.md">中文</a> | <a href="README_en.md">English</a></h3>

> 从 `<h1>Hello World</h1>` 到搭建一个完整的个人主页

## 这是什么？

一套给**想做网页但不知道从哪开始**的人准备的课程。

不是那种"我给你看代码你自己悟"的课——每节课都有**可交互的示例、练习题、生活类比**，15 课带你从写第一个 HTML 标签到能自己做一个完整的个人主页（带 CSS 样式、JavaScript 交互、响应式布局）。

每课的 .html 文件直接用浏览器打开就能看到效果，不需要装任何东西。学到最后你会明白：HTML 是骨架，CSS 是皮肤，JavaScript 是灵魂。

## 怎么跑？

```bash
# 进入课程目录
cd "d:/VS project/Mimo project/s_html"

# 直接用浏览器打开任意一课
# 或者在 VS Code 中安装 Live Server 扩展，右键 "Open with Live Server"
```

## 学什么？

| 02 | 文章排版 | 标题、段落、引用、格式化 |
| 03 | 图文并茂 | 超链接、图片、图片映射 |
| 04 | 列表 + 基础样式 | ul/ol/dl、内联 CSS |
| 05 | 数据表格 | table、合并单元格、表格分区 |
| 06 | 用户表单 | input、select、datalist、验证 |

### 第二阶段：CSS 样式 🎨
开始装修。颜色、字体、布局、动画——让网页从"能看"变成"好看"。

| # | 你会做什么 | 顺便学了什么 |
|---|-----------|-------------|
| 07 | 网页美化 | 颜色、字体、背景、盒模型 |
| 08 | 精准选元素 | 选择器、伪类、伪元素 |
| 09 | 页面布局 | Flexbox、Grid、定位、媒体查询 |

### 第三阶段：JavaScript 交互 ⚡
注入灵魂。变量、函数、DOM、事件——让网页能"思考"和"反应"。

| # | 你会做什么 | 顺便学了什么 |

### 第四阶段：项目实战 🎯
把前面学的全串起来。做一个完整的个人主页。

| # | 你会做什么 | 用到了什么 |

## 几个小建议

1. **别跳课** — 后面的课用到前面的知识，跳了会懵
2. **别光看** — 直接用浏览器打开，点击、输入、交互
3. **做练习** — 练习题不是摆设，先自己写再看答案
4. **按 F12** — 打开浏览器开发者工具，查看控制台输出
5. **改代码** — 修改代码中的值，看效果怎么变

## 环境要求

- **浏览器** — Chrome、Firefox、Edge 任意一个
- **编辑器** — 推荐 VS Code + Live Server 扩展

## 课程特色

- 🎯 **生活类比** — 每课都有，把抽象概念变成日常例子
- 📝 **逐行注释** — 每行代码都有详细说明
- 🖱️ **可交互** — 每个示例都可以点击、输入、操作
- ⚠️ **常见陷阱** — 标注容易犯的错误，避免踩坑
- 📋 **练习题** — 每课 3 道，有题目+提示+参考答案
- 🎉 **恭喜完成** — 每课都有正反馈
