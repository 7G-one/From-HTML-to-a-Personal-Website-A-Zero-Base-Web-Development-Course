# 🌐 HTML/CSS/JavaScript Course

<h3 align="center"><a href="README.md">中文</a> | <a href="README_en.md">English</a></h3>

> From `<h1>Hello World</h1>` to building a complete personal website

## What is this?

A course for **people who want to build websites but don't know where to start**.

Not the kind where "I show you code and you figure it out" — every lesson has **interactive examples, exercises, and real-life analogies**. 15 lessons take you from writing your first HTML tag to building a complete personal website with CSS styling, JavaScript interactions, and responsive design.

Each .html file can be opened directly in your browser — no installation needed. By the end, you'll understand: HTML is the skeleton, CSS is the skin, JavaScript is the soul.

## How to run?

```bash
# Enter the course directory
cd "d:/VS project/Mimo project/s_html"

# Open any lesson in your browser
# Or install the Live Server extension in VS Code, right-click "Open with Live Server"
```

## What will you learn?

### Stage 1: HTML Basics 🏗️
Learn to build the skeleton. Tags, text, links, images, lists, tables, forms — the structure of web pages.

| # | You'll Build | Along the Way |
|---|-------------|---------------|
| 00 | Your first webpage | HTML tags, text formatting |
| 01 | Complete page structure | DOCTYPE, head, body, meta |
| 02 | Article layout | Headings, paragraphs, quotes |
| 03 | Images and links | Hyperlinks, images, image maps |
| 04 | Lists + basic styles | ul/ol/dl, inline CSS |
| 05 | Data tables | table, merged cells, table sections |
| 06 | User forms | input, select, datalist, validation |

### Stage 2: CSS Styling 🎨
Start decorating. Colors, fonts, layouts, animations — make web pages go from "functional" to "beautiful".

| # | You'll Build | Along the Way |
|---|-------------|---------------|
| 07 | Page beautification | Colors, fonts, backgrounds, box model |
| 08 | Precise element selection | Selectors, pseudo-classes, pseudo-elements |
| 09 | Page layouts | Flexbox, Grid, positioning, media queries |
| 10 | Animations | transition, @keyframes |

### Stage 3: JavaScript Interactions ⚡
Inject the soul. Variables, functions, DOM, events — make web pages "think" and "react".

| # | You'll Build | Along the Way |
|---|-------------|---------------|
| 11 | JS core | Variables, functions, objects, arrays, DOM |
| 12 | Event handling | Event delegation, keyboard events, preventDefault |
| 13 | Responsive design | Media queries, rem/vw/clamp |

### Stage 4: Project 🎯
Put it all together. Build a complete personal website.

| # | You'll Build | Uses |
|---|-------------|------|
| 14 | Personal website | HTML + CSS + JavaScript |

## A few tips

1. **Don't skip lessons** — Later lessons use knowledge from earlier ones
2. **Don't just read** — Open in browser, click, type, interact
3. **Do the exercises** — They're not decoration, try them yourself first
4. **Press F12** — Open browser DevTools, check the console
5. **Modify code** — Change values and see what happens
